# FAQ and common issues

## I would like to contribute to simulation. How do I do that in practice?

Any help would be greatly appreciated. A series of tasks are listed on [the Simulation tasks GitHub project page](https://github.com/orgs/ShipSoft/projects/11), and will evolve along with the needs of simulation. If you are unsure, join the simulation mattermost channel, the simulation meetings or send a message to Hanae.

## I am using FairShip and would like to use Aegir, but cannot right now because something is missing.

Some bits and pieces still need to be added. If a feature which you would like to have is missing, report it as an issue, in the mattermost channel or at the simulation meeting.

## What is "aegir"?

Ægir is a personification of the sea in Norse mythology.
